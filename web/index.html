📋 Aide — Météo de mes villes

🌡️ La grille principale

Chaque ligne = une ville, chaque colonne = un jour. Les cases affichent la température max (grand) et min (petit), avec un code couleur du bleu (froid) au rouge (chaud). En bas de chaque case : les rafales de vent 💨, la pluie cumulée 🌧 (si > 0) et l'humidité minimale %.

🔍 Prévisions heure par heure

Pour chaque ville, trois petits boutons permettent d'ouvrir une vue détaillée heure par heure :

  • A = AROME (modèle français, ~2-3 jours, très précis)
  • P = ARPEGE (modèle français global, ~3-4 jours)
  • D = ICON-D2 (modèle allemand, ~2-3 jours, haute résolution)

Si un bouton est grisé, le modèle n'a pas de donnée pour cette ville.

➕ Ajouter une ville

Dans la barre du haut, colle un lien Météociel (n'importe quelle page de la ville) puis clique sur ＋ Ajouter. Exemples valides :

  • https://www.meteociel.fr/previsions/10232/lanuejols.htm
  • https://www.meteociel.fr/previsions-arome-1h/28366/rouen.htm

L'app extrait automatiquement l'identifiant et le nom de la ville. La ville est sauvegardée et reviendra à chaque lancement.

🗑️ Supprimer une ville

Clique sur le bouton rouge × à côté du nom d'une ville. Une confirmation te sera demandée.

🔄 Actualiser

Le bouton Actualiser re-télécharge toutes les prévisions. À noter : à chaque lancement de l'app, tout est automatiquement re-téléchargé (zéro cache).

📊 Les modèles météo

L'app combine plusieurs sources de Météociel pour maximiser la couverture :

  • GFS (prévisions + tendances) : ~11 jours, global — alimente la grille principale
  • AROME : modèle régional français, haute résolution, idéal pour le court terme
  • ARPEGE : modèle global français, bonne couverture moyenne
  • ICON-D2 : modèle régional allemand, bon complément

Comparer ces modèles te donne une idée de l'incertitude : s'ils sont d'accord, la prévision est fiable ; s'ils divergent, prudence.

💾 Où sont mes données ?

Ta liste de villes est sauvegardée dans %APPDATA%\MeteoVilles\villes.json. Tu peux l'éditer manuellement si besoin. Aucune autre donnée n'est stockée.
